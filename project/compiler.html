Inside Effects

Intro Language Effects Analysis Compiler

As an experiment, a source to source compiler was implemented that translates from the "inside" language to JavaScript. Eventually the compiler should be able to compile itself, however it is currently not yet feature complete.

In general, the compiler was implemented in a functional style without side effects. This makes sense because the compiler at this point doesn't need to maintain a lot of state. However, some of the helper functions such as map and concat take advantage of the fact that arguments are call by value and simply mutate their arguments instead of creating a new variables.
